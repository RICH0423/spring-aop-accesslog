## To get a Git project into your build:

- Add the JitPack repository to your build file
```xml
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

- Add the dependency
```xml
<dependency>
    <groupId>com.github.RICH0423</groupId>
	  <artifactId>spring-aop-accesslog</artifactId>
	  <version>Tag</version>
</dependency>
```
